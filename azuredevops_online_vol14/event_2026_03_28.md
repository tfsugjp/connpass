### GitHub Advanced Security × Defender for Cloudで開発とSecOpsのサイロを超える: コードとクラウドをつなぐ、開発プラットフォームのセキュリティ

GitHub Advanced Security（GHAS）は、開発者の近くでセキュリティを実装できる強力な仕組みですが、
「実行環境で何が起きているか」「クラウド全体のリスクをどう捉えるか」という観点では、
GitHub単体ではカバーしきれない領域も存在します。

本セッションでは、Defender for Cloud と GitHub を連携することで、GHAS のセキュリティをどのように拡張できるのか、
そして 開発（Dev）と運用・セキュリティ（SecOps）の分断をどう超えられるのか、DevSecOpsの導入・運用で発生しがちな課題、そしてどのように立ち向かうかというストーリーを実体験を交えてご紹介します。
